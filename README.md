# Infinite-Monkey Simulation 🙊,🙉,🙈

🐒 The Infinite Monkey Theorem
A monkey hitting keys at random for an infinite amount of time will almost surely type any given text — like the complete works of Shakespeare. This simulation lets you watch the idea play out, and shows you exactly why "almost surely" and "before the heat-death of the universe" are two very different promises.

Source: [Infinite Monkey Theorem](https://en.wikipedia.org/wiki/Infinite_monkey_theorem)
